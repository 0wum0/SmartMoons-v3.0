Test der Mobile Sidebar Navigation

Diese Testseite demonstriert die funktionierende Sidebar-Navigation für SmartMoons.

  • Desktop (>768px): Sidebar ist links fixiert und immer sichtbar
  • Mobile (≤768px): Sidebar ist versteckt, Burger-Button öffnet sie

Features

  • ✅ Smooth Slide-Animation
  • ✅ Overlay mit Klick zum Schließen
  • ✅ Sci-Fi Design mit Glow-Effekten
  • ✅ Responsive für alle Bildschirmgrößen
  • ✅ Touch-optimiert für Mobile
  • ✅ Accessibility-Features (ARIA)

Interaktivität

Teste die folgenden Aktionen:

  1. Klicke auf den Burger-Button (☰) um die Sidebar zu öffnen
  2. Klicke auf das Overlay oder einen Link um sie zu schließen
  3. Drücke ESC um die Sidebar zu schließen
  4. Verändere die Fenstergröße um Desktop/Mobile-Ansicht zu testen
