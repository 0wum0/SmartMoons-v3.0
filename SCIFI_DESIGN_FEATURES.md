```css

    100% { transform: translateX(100%); }
}
```
- Verwendet auf: Progress Bars
- Dauer: 2 Sekunden

```css

    0%, 100% { box-shadow: 0 0 5px cyan; }
    50%      { box-shadow: 0 0 20px cyan; }
}
```

- Dauer: 2 Sekunden
