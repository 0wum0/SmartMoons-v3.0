Test-Funktionen

Testen Sie die folgenden Funktionen:

  1. Klicken Sie auf das linke Burger-Menü (☰) - die Sidebar sollte von links hereingleiten
  2. Klicken Sie auf das rechte Burger-Menü (☰) - die Sidebar sollte von rechts hereingleiten
  3. Klicken Sie auf den dunklen Overlay-Bereich - das Menü sollte sich schließen
  4. Klicken Sie auf den X-Button in der Sidebar - das Menü sollte sich schließen
  5. Drücken Sie die ESC-Taste - alle offenen Menüs sollten sich schließen
  6. Die Burger-Icons animieren sich zu einem X wenn aktiv
  7. Der Seitenhintergrund wird abgedunkelt wenn ein Menü offen ist

Responsive Design

Das Interface passt sich automatisch an verschiedene Bildschirmgrößen an:

  • Mobile (< 768px): Burger-Menü aktiv, Sidebars versteckt
  • Tablet (768px - 1200px): Optimierte Layouts
  • Desktop (> 1200px): Sidebars statisch sichtbar, kein Burger-Menü

Diese Test-Seite zeigt immer die Mobile-Ansicht für Demonstrationszwecke.

Zugänglichkeit

Implementierte Zugänglichkeitsfunktionen:

  • aria-expanded zeigt den Status des Burger-Menüs an
  • aria-hidden versteckt geschlossene Menüs vor Screenreadern
  • aria-label beschreibt die Funktion der Buttons
  • aria-controls verknüpft Buttons mit ihren Zielelementen
  • Tastatur-Navigation mit ESC-Taste
  • Fokus-Management beim Öffnen/Schließen
